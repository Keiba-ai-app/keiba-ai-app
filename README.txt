競馬分析アプリ

同梱ファイル:
- index.html
- manifest.json
- sw.js
- README.txt

使い方:
1. ZIPを解凍します。
2. index.html をブラウザで開きます。
3. タブ区切りデータを貼り付けます。
4. 距離・馬場・回り・競馬場を必要に応じて選びます。
5. 「分析実行」を押します。

補足:
- manifest.json を入れてあるので、環境によってはホーム画面追加に対応します。
- PWAとして完全動作させるには、通常はWebサーバー経由で開く必要があります。
- 単純に使うだけなら index.html を直接開けばOKです。
